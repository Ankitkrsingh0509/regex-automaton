① Thompson Construction ε-NFA
② ε-NFA → NFA (ε-Elimination)
Algorithm: For each state, compute its ε-closure. For each symbol a in the alphabet and each state q, the new NFA transition δ'(q, a) = ε-closure(δ(ε-closure(q), a)). A state is accepting if its ε-closure contains any original accept state.
③ NFA → DFA (Subset Construction / ε-Closure)
Closure Method: Starting from the ε-closure of the start state, for each DFA state (a set of NFA states) and each symbol, compute move() then ε-closure(). If a new set of NFA states is found, create a new DFA state. Repeat until no new states appear.
④ Equivalence Method (Partition Refinement)
Equivalence Method: Start with two partitions — accepting and non-accepting states. Repeatedly refine: two states in the same partition are split if they transition to different partitions on any symbol. When stable, each partition becomes one minimal-DFA state.

⑤ Myhill-Nerode Method (Table-Filling)
Myhill-Nerode Theorem: Two states are distinguishable if one is accepting and the other is not, or if on some input symbol they transition to distinguishable states. Build a triangular table, mark distinguishable pairs. Unmarked pairs are equivalent and can be merged.
